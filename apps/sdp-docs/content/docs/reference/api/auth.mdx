---
title: Auth
description: Session authentication and management.
---

| Method | Endpoint | Summary |
| --- | --- | --- |
| `POST` | `/v1/auth/logout` | Logout session |
| `GET` | `/v1/auth/me` | Get current user |
| `GET` | `/v1/auth/sessions` | List sessions |
| `DELETE` | `/v1/auth/sessions/{sessionId}` | Revoke session |
